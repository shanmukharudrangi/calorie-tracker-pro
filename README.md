# CalorieTracker Pro
A responsive fitness web application that calculates daily calorie requirements using the Mifflin-St Jeor BMR formula, provides BMI analysis, macro nutrient breakdown visualization, and workout calorie estimation with dark mode support.

## Features
- Daily Calorie Calculator
- BMI Calculator
- Macro Nutrient Breakdown
- Workout Calorie Burn Estimator
- Home Workout Library
- Dark Mode Support
- Fully Responsive Design

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
